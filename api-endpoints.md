# API Endpoints

### endpoints

- Teams
  - /api/v1/teams/list - get all teams
  - /api/v1/teams/get - get specific teams
  - /api/v1/teams/create - create a team
  - /api/v1/teams/edit - modify a team
  - /api/v1/teams/delete - delete a team
- Users
